# 样例MOD：金书红颜录5 如画江山

本目录是一个新的 `群侠传，启动！` 样例MOD，MOD ID 为 `jshyl`。

目标是把 [TPRHandbook](https://github.com/inkitter/TPRHandbook) 中的《金书红颜录5 如画江山》5.52j 路线手册，逐步转化为 JYX2 框架中的可游玩内容。

## 当前实现

- `ModSetting.asset`：让编辑器和MOD启动器识别 `金书红颜录5 如画江山`。
- `Configs/`：以 JYX2 配置表为种子，沿用现有角色、物品、武功、战斗、场景表结构。
- `Lua/jshyl_story_index.lua`：记录手册路线与章节索引，作为后续事件脚本拆分的来源地图。
- `Lua/jshyl_route_data.lua`：整理《鸳鸯刀》和《越女剑》的事件节点、前置条件、战斗、奖励与状态变化。
- `Lua/0.lua`：可触发的开局事件，用于验证新游戏进入地图后能够执行 `jshyl` 的 Lua 内容。
- `Lua/modentry.lua`：MOD 初始化入口，验证路线索引已随 MOD 预加载。
- `Maps/GameMaps/00_mochuanlinju.unity`：复用样例MOD中的小型起始场景，作为路线系统纵切的临时开局地图。

## 后续路线

1. 把《鸳鸯刀》《越女剑》的 `jshyl_route_data` 节点接到实际地图触发器。
2. 为路线节点补齐战斗配置、奖励发放、队友加入/离队和路线状态保存。
3. 把手册中的 `战斗`、`入队/离队`、`属性变化`、`重要条件` 映射为可验证的事件指令。
4. 按书系扩展：主角剧情、飞狐、天龙、射雕、鹿鼎、笑傲、书剑、神雕、侠客、倚天等。
5. 最后接入终战与多路线结局条件。

这个MOD应尽量保持为内容层实现；只有当路线系统确实缺少通用能力时，才考虑修改共享框架。
